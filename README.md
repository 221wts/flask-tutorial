使用Python和Flask开发你的第一个Web程序。

## 特点

* 基于Flask 1.0
* 使用一个在线记事本程序作为示例
* 复原完整的开发流程，提供每一个命令和代码块
* 读者需要从头到尾，自己跟着敲出每一个代码，通过模仿来学习Flask
* 只提供入门所需的最少信息
* 术语优化
